# sklearnX

## Overview
**sklearnX** is a lightweight machine learning library built from scratch in Python.  
It implements core machine learning algorithms using **NumPy** while maintaining a clean and modular architecture inspired by **scikit-learn**.

The goal of this project is to understand how machine learning algorithms work internally by implementing them from first principles while keeping a structure similar to modern machine learning libraries.

---

## Implemented Algorithms
- K-Nearest Neighbors (KNN) Classifier
- K-Nearest Neighbors (KNN) Regressor

---

## Key Features
- Machine learning algorithms implemented from scratch
- Modular and extensible project structure
- Distance-based learning algorithms implemented using NumPy
- Clean design inspired by scikit-learn library structure
